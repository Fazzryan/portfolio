Tentang Saya

Halo! perkenalkan nama saya Dinda Fazryan, Saya masih kuliah semester 6 di STMIK Tasikmalaya jurusan Teknik Informatika. Saya tertarik dengan dunia IT sejak SMA. Karena di SMA tidak ada pelajaran IT, saya pun terpaksa belajar otodidak dari youtube. Chanel youtube yang sering saya lihat yaitu Web Programming Unpas.

Saat ini saya menguasai HTML, Css, Bootstrap, sedikit JavaScript, PHP dan Android Studio berkat belajar otodidak dari internet dan youtube. Setelah masuk perkuliahan, saya juga sedikit menguasai bahasa pemrograman yang lain seperti C, C++, Java, dan Vb.net. Follow sosial media saya dibawah ini, terima kasih.
